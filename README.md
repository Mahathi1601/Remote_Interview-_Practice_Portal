# 🚀 Remote Interview Practice Portal

### *Practice Smart. Perform Better.*

---

## 📌 Overview

A web platform to help users prepare for interviews through text-based practice, video recording mock interviews, and performance tracking.

---

## 🎯 Key Features

- 📝 Text Practice – Answer questions with AI feedback and scoring
- 🎥 Video Mock Interview – Record answers and review later
- 📊 Confidence Tracking – Rate your confidence level after each answer
- 📈 Dashboard – Track scores, charts, and progress over time
- 👤 Profile Management – Manage your personal details and skills
- 📂 Interview History – Store and review past sessions

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js with Express.js
- Database: MongoDB with Mongoose ODM
- Authentication: JWT and bcryptjs
- File Upload: Multer for video recordings

---

## ⚙️ How It Works

Step 1: Login or Register to your account
Step 2: Choose between Text Practice Mode or Video Mock Interview Mode
Step 3: Select an interview category (Technical, HR, or Aptitude)
Step 4: Answer questions within the time limit
Step 5: For text answers, get instant AI feedback and score
Step 6: For video answers, record yourself and rate your confidence
Step 7: Review all your progress on the Dashboard
Step 8: Watch back your recorded answers anytime from Profile

---

## 📊 Problem Solved

Most platforms only provide interview questions without proper simulation. This platform adds time-based practice, video recording for self-review, confidence measurement, and complete performance tracking.

---

## 💡 Advantages

- Free and accessible to everyone
- Realistic interview experience with timers

## 📌 Quick Setup Guide

Step 1: Install Node.js from nodejs.org
Step 2: Install MongoDB locally or create free MongoDB Atlas account
Step 3: Open backend folder and run npm install
Step 4: Create .env file with PORT, MONGODB_URI, and JWT_SECRET
Step 5: Run npm start to start the backend server
Step 6: Open frontend HTML files directly or use Live Server extension
Step 7: Register a new account and start practicing
